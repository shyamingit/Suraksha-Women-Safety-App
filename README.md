# 🚨 Suraksha – Women Safety Android App

🚀 Suraksha is a real-time women safety application designed to provide instant emergency response, live location tracking, and smart monitoring using health and environmental signals.

---

## 🎯 Problem Statement

Many women face safety risks in public spaces, especially when traveling alone.  

Current challenges include:
- Lack of immediate emergency response  
- Inability to share real-time location quickly  
- No intelligent system to detect danger situatios  
- Limited awareness of unsafe areas  

This leads to delayed help and increased vulnerability during emergencies.

---

## ✅ Solution

Suraksha addresses these challenges by providing a **smart, real-time safety system** that:

- Instantly shares live location with trusted contacts  
- Allows quick SOS activation during emergencies  
- Detects abnormal stress or health conditions  
- Provides awareness of unsafe (danger) zones  

---

## 🚀 Key Features

### 📍 Live Location Sharing  
- Real-time loation tracking with family members  
- Continuous monitoring for safety assurance  

### 🚨 SOS Emergency Alert  
- One-tap / long-press SOS activation  
- Instantly shares live location  
- Triggers emergency communication  

### 📷 Live Camera Feed  
- Shares real-time surroundings during emergencies  
- Provides visual context for quicker response  

### ❤️ Smart Health Monitoring  
- Tracks heart rate via smartwatch (if available)  
- Detects abnormal stress levels  
- Sends alerts when irregular actvity is detected  
- Can help in critical situations like sudden health issues  

### ⚠️ Danger Zone Detection  
- Marks unsafe areas as Red Zones  
- Identifies safer areas as Green Zones  
- Helps users avoid high-risk locations
